# Documentation

This folder contains documentation and example files for the EvMotorsScanner app.

## Example Files

### AssetManager.example.js
Contains usage examples and best practices for the AssetManager utility:
- How to use images, fonts, colors, and common styles
- Font configuration with expo-font
- Dynamic asset access
- Theme-based styling
- Custom styling examples

### HeaderWithLogout.example.js
Contains usage examples for the HeaderWithLogout component:
- Basic navigation setup
- Custom titles and styling
- Conditional logout buttons
- Multiple screen configurations
- Screen component integration

## Usage

These example files are for reference only and should not be imported into your main application code. They demonstrate how to properly use the various utilities and components in the EvMotorsScanner app.

## Getting Started

1. Review the example files to understand how to use the utilities
2. Copy relevant code patterns into your own components
3. Customize the examples to fit your specific needs
4. Follow the patterns shown for consistent code structure

## Best Practices

- Always use the AssetManager for consistent styling
- Use the HeaderWithLogout component for authenticated screens
- Follow the established patterns for navigation setup
- Keep your code organized and maintainable
